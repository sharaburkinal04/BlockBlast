# Block Blast — курсовая работа

## Описание проекта
Классическая игра-головоломка «Block Blast», разработанная на Kotlin с использованием фреймворка Jetpack Compose для платформы Android.

Игрок перетаскивает фигуры на игровое поле 8×8, заполняя строки и столбцы для получения очков. Проект включает генерацию 14 типов фигур, глобальную таблицу рекордов через Firebase, поддержку тёмной и светлой темы, локализацию на русский и английский языки, звуковое сопровождение.

## Основные возможности
- **Игровой процесс**: перетаскивание фигур, заполнение строк и столбцов, начисление очков.
- **Генератор фигур**: 14 форм.
- **Глобальная таблица рекордов**: топ-10 лучших результатов через Firebase.
- **Интерфейс**: две темы (светлая/тёмная) и локализация (русский/английский).
- **Звуковое сопровождение**: фоновая музыка, звуки игровых действий.
- **Экраны**: «Game Over» (итоговый счёт, рестарт) и «Настройки» (громкость, язык, тема).
- **Сохранение**: настройки сохраняются через SharedPreferences.

## Инструкция по запуску
1. Убедитесь, что у вас установлена Android Studio (версия Hedgehog или выше).
2. Клонируйте репозиторий.
3. Откройте проект в Android Studio (Open → выберите папку проекта).
4. **Подключите Firebase**:
    - Скачайте файл `google-services.json` из консоли Firebase.
    - Поместите его в папку `app/`.
5. Соберите и запустите проект (подключите устройство или эмулятор и нажмите `Run`).

## Управление игровым процессом
- **Выбор**: нажатие на фигуру.
- **Размещение**: перетаскивание фигуры на поле (drag-and-drop).

## Зависимости
Проект использует следующие компоненты:
- Kotlin
- Jetpack Compose (Material 3)
- Firebase Realtime Database
- Coroutines
- MediaPlayer
- SharedPreferences

*Полный список указан в файле `build.gradle.kts (Module: app)`.*
